# Task Management System

## Prerequisites
- Docker installed
- Docker Compose installed

## Running the project

1. Open Docker Desktop, then build and start the Docker containers in the terminal of Visual Studio Code: docker-compose up --build

2. Access the application through address bar on the browser: http://localhost:3000

3. To stop the containers: Use the key combination (Ctrl + C) 3 times in the terminal of Visual Studio Code.

## Testing
- Create a new task using the frontend interface or by sending a POST request to the backend `/api/tasks` endpoint.
- View, edit, or delete tasks using the provided frontend controls or through backend API calls.
